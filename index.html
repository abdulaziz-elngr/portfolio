نبذة عني

أنا مهندس أنظمة مدمجة وإنترنت أشياء (IoT) ومطور ويب متكامل، شغوف بربط العالم المادي بالرقمي وبناء حلول ذكية ومتكاملة. تخصصي يجمع بين تصميم الأنظمة الصلبة (Hardware) القائمة على المتحكمات الدقيقة وتطوير البرمجيات (Software) لإدارة البيانات عبر الويب وتطبيقات الهاتف.

أمتلك خبرة عملية في التعامل مع المتحكمات الدقيقة مثل ESP32 وArduino وتطوير لوحات تحكم متقدمة لمراقبة البيانات لحظياً باستخدام تقنيات الويب الحديثة والـ بروتوكولات الصناعية. أسعى دائماً لتطبيق معايير الكود النظيف والهندسة المنظمة لتحويل الأفكار الإبداعية إلى مشاريع حقيقية ذات أثر صناعي وتكنولوجي.

المشاريع الحالية والهندسية

نظام مكافحة الحرائق الذكي ولوحة التحكم (Smart Home Firefighting)

نظام متكامل يعتمد على متحكم ESP32 ومجموعة من مستشعرات الغاز واللهب الذكية لرصد وتوقع الحرائق وتسريب الغازات، مع بناء لوحة تحكم متقدمة (Dashboard) باستخدام C++ وQt/QML لمراقبة البيانات والتحكم في النظام لحظياً عبر الاتصال التسلسلي واللاسلكي.

ESP32 C++ Qt/QML Sensors Integration

روبوت مكافحة الحرائق الذاتي القيادة (Autonomous Firefighting Robot)

تصميم وبرمجة روبوت ذكي يعتمد على برمجيات الأنظمة المدمجة، يقوم بالتحرك الذاتي داخل المنشآت للكشف عن بؤر النيران باستخدام مستشعرات اللهب، والتوجه نحوها تلقائياً لتفعيل نظام الإطفاء الذاتي والميكانيكي مع إرسال إشعارات فورية.

Arduino / ESP32 Embedded C Robotics Hardware Design

نظام إدارة الابتكار الجامعي (Innovation Management System - IMS)

نظام ويب متكامل لإدارة ومراقبة موارد الجامعة من قاعات، ومختبرات علمية، ومكتبات رقمية، يتيح للطلاب وأعضاء هيئة التدريس تتبع وحجز الموارد بكفاءة وسهولة تامة عبر قاعدة بيانات مركزية.

HTML5 / CSS3 JavaScript PHP MySQL

الموقع التعريفي الشخصي (Cyber Portfolio)

تصميم وتطوير هذا الموقع التعريفي المتجاوب باستخدام أحدث أساليب التصميم الزجاجي والمظلم (Glassmorphism) لاستعراض المهارات الهندسية والمشاريع التقنية بشكل احترافي.

UI/UX Design Glassmorphism Responsive Web
