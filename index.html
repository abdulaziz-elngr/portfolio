About Me

أنا مهندس أنظمة مدمجة وإنترنت أشياء (IoT) ومطور ويب متكامل، شغوف بربط العالم المادي بالرقمي وبناء حلول ذكية ومتكاملة. تخصصي يجمع بين تصميم الأنظمة الصلبة (Hardware) القائمة على المتحكمات الدقيقة وتطوير البرمجيات (Software) لإدارة البيانات عبر الويب.

أمتلك خبرة عملية في التعامل مع المتحكمات الدقيقة مثل ESP32 وArduino وتطوير أنظمة ويب قوية باستخدام تقنيات الويب الحديثة. أسعى دائماً لتطبيق معايير الكود النظيف والهندسة المنظمة لتحويل الأفكار الإبداعية إلى مشاريع حقيقية ذات أثر صناعي وتكنولوجي.

I am an Embedded Systems & IoT Engineer and a Full-Stack Web Developer, passionate about connecting the physical world with the digital one to build integrated smart solutions. My expertise spans across hardware design using microcontrollers and software development for data management over the web.

I have hands-on experience with microcontrollers like ESP32 and Arduino, alongside building robust web systems using modern web technologies. I strive to apply clean code standards to transform creative ideas into real-world projects with a tangible technological impact.
